## Description

このリポジトリには以下が含まれる
* パーツ分けした3Dモデル
* パーツ組み立てのスクリプト

## Requirement

動作確認はPython 3.12.1を使用  
[Harbor3d](https://github.com/MarataUni/Harbor3d) version 0.2.5がインストールされている環境で実行できる

## Usage

1. _parts_reversed/reverse.pyを実行
1. _parts_renamed/rename_parts.pyを実行
1. modeling_posed.pyを実行
